# baiduhtml
### 前端之旅 ###   

**DAY1**   再一次从零开始学习前端，摆脱之前三天打鱼两天晒网的坏习惯，认证坚持到最后，半个月之类走遍百度前端技术学院，能够彻底掌握课程内容知识点并且学会合理运用到实际项目中，二十天一之内对技术进行一个阶段的提升，并且拿到一份相对满意的offer。   

**DAY2**   从早上起床到现在全身提不起力气，对认识事情都没有一点兴趣，部分原因是受天气影响，大部分因素还是态度没有端正过来，一坐到电脑跟前，潜意思就告诉自己休息一下调整一下状态没关系的，而且拖延思维也一直影响着自己，总告诉自己今天的任务往后面挪一天也是可以完成的，没什么的，调整一下状态才能更好的进行之后的任务，也有另一个思维告诫自己不可以这样子，但是最后总是懒惰贪玩的想法压倒了其他想法，学习真的好难，在自己不想学时候强行去学更难，强迫自己的时候就像是有无数只蚂蚁在自己身上爬，心里难受的不行，明明该继续学下去，态度要放好，但是就是没法做到，写完这些的时候又突然有点急躁，而且一天还是虚度了，希望这种状态越来越少，后续能够控制消除吧。   

**DAY3**   前端之旅第三天，浏览了一下其他优秀同学的作业，对自己的简历的美化有了一点想法，不过依旧没有付诸现实，完成了掘金的部分任务，虽然有点粗制滥造，但是也接触到了些许之前并未掌握的知识，对部分知识的了解以及运用得到加固，后面更多的改进在于想法立即实现，不要拖延，并且代码方面更多的手动，而非复制其他人的代码。   

**DAY4**   前端之旅第四天，今天什么事情也没做，掘金任务都没有完成，愈发的差劲了，每次大脑中列举的计划都不愿意去实际实施，即将一个周过去了，二十一天也没剩多少了，技术没有任何提升倒是胡思乱想了很多，就写这么多吧，明天加油，没有时间再贪玩了，也不是可以任性的年纪了
